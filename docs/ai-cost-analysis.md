## 推奨APIプロバイダー

### 主軸: Replicate API

- 画像アップスケーリング、カラー化、動画アップスケーリング、背景除去（RMBG-2.0）はすべてReplicate APIで対応可能
- 従量課金で管理しやすい
- 複数モデルを統一インターフェースで利用できる

### 補完: Remove.bg API

- 背景除去の精度が特に重要なユースケース（例: Pro向けオプション）で検討
- 通常用途はReplicate (RMBG-2.0) で十分

---

## リスク要因

### 1. API料金変動

- Replicate・Remove.bgの料金は予告なく変更される可能性がある
- 定期的なコスト見直し（四半期ごと推奨）が必要

### 2. 為替レート変動（USD/JPY）

- API料金はUSD建てのため、円安が進むと実質コストが上昇する
- 現在の試算は概算値。為替変動±10%程度のバッファを持った価格設定を推奨

### 3. 高負荷ユーザー

- 特にLifetimeプランで、想定以上に利用するユーザーが発生するリスク
- フェアユースポリシーまたは月間上限の設定で対策
