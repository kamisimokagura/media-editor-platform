# Bundled Fonts

This directory contains self-hosted webfonts used by `next/font/local`.

- Inter (`inter-latin-wght-normal.woff2`) from @fontsource-variable/inter
- Poppins (`poppins-latin-400..900-normal.woff2`) from @fontsource/poppins

These packages source files from Google Fonts and are distributed under SIL Open Font License 1.1.
